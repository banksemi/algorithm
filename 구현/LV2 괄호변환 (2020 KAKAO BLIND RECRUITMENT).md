카카오 문제

이번도 구현 문제이다.

그러나 문제가 상당히 복잡하게 서술되어있다. 그대로 따라하고 문제가 해결되었다.

```
"""
균형잡힌 괄호 문자열: (랑 ) 개수가 동일
올바른 괄호 문자열: 잘 맞춰짐.


1. 빈 문자열 -> 빈 문자열 그대로 반환
2. w를 '()개수가 맞는' 2개의 문자열로 분리
- 단 u는 ()개수가 맞는 괄호 문자열로 더 분리를 할 수 없어야함.
- u를 고치고 나서 v도 똑같이 고치고 붙여서 반환해야함.

- u가 괄호 짝이 안맞으면
- 빈 문자열에 첫번째 

"""

def function(w):
    # 1. 입력이 빈 문자열인 경우, 빈 문자열을 반환합니다.
    if len(w) == 0:
        return w

    # 2. 문자열 w을 두 균형잡힌 괄호 문자열로 분리합니다.
    # u는 더 이상 분리가 안되어야합니다. -> 즉 조건을 만족하자마자 u, v를 확정합니다.
    count = 0
    correct = True
    u = ''
    for i in w:
        if i == '(':
            count += 1
        else:
            count -= 1
        u += i
        # 음수가 되는 경우 올바른 괄호 문자열이 아님.
        if count < 0:
            correct = False
        if count == 0:
            break
    v = w[len(u):]
    # 3. 문자열 u가 올바른 괄호 문자열이라면, v에 대해 1단계부터 다시 수행하고, 결과를 u에 붙여서 반환합니다.
    if correct:
        return u + function(v)
    # 4. u가 올바른 괄호 문자열이 아니라면
    temp = '(' # 4.1 빈 문자열에 (을 붙입니다.
    temp += function(v) # 4.2 v 에 대한 재귀적 수행 결과를 이어 붙입니다.
    temp += ')' # 4.3 )을 다시 붙입니다.
    # u의 첫 번째와 마지막 문자를 제거하고, 나머지 문자열의 괄호 방향을 뒤집어서 뒤에 붙입니다. 
    for i in range(1, len(u) - 1):
        if u[i] == '(':
            temp += ')'
        else:
            temp += '('

    return temp

def solution(p):
    answer = ''
    return function(p)
```
