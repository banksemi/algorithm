문제를 보고 조합을 떠올려서 2 ** 10000 이라고 생각했던 문제.

결국 답을 봤다.

그냥 직관적으로 생각해볼껄.

내가 카드를 정리한다고 치면 카드를 보통 가로로 눕히거나 세로로 눕힐거다.


그 논리로 풀면 n으로 완료된다.
```
def solution(sizes):
    # 일단 카드들을 가로로 눕힌다.
    
    c = []
    for w, h in sizes:
        if w > h:
            c.append((w, h))
        else:
            c.append((h, w))
    
    return max([i[0] for i in c]) * max([i[1] for i in c])
```
