문제 조건과 예시를 통해 규칙을 찾아야하는 문제.

'110'이라는 단어를 추출해야하는데, 추출 과정에서 새로운 '110'이 만들어질 수 있으므로 스택 구조를 사용했다.

테스트 케이스, 최악의 케이스를 모두 테스트해보자.
참고: pop()을 2번 쓰는 것도 방법이지만 `del arr[-2:]` 도 가능하다.

def extract(s):
    count = 0
    stack = []
    
    for i in s:
        if i == '0':
            # 꺼낼 수 있는지 본다.
            if len(stack) >= 2 and stack[-1] == '1' and stack[-2] == '1':
                stack.pop()
                stack.pop()
                count += 1
            else:
                stack.append(i)
        else:
            stack.append(i)
    return count, ''.join(stack)

def make_result(count, word):
    candidate = 0 # 0번 앞에 넣기
    found_1 = 0
    for i, char in enumerate(word):
        if char == '0':
            candidate = i+1
            found_1 = 0
        else:
            found_1 += 1
        if found_1 == 3:
            break
    return word[0:candidate] + '110' * count + word[candidate:]

def solution(s):
    # s = ["1" * 700_000 + '0' * 300_000]
    answer = []
    for original_word in s:
        count, word = extract(original_word)
        answer.append(make_result(count, word))
    return answer
