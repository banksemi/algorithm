```
"""
N개의 서로 다른 숫자가 있는 배열 A
- 1 이상, N+1 이하의 숫자 있으며, 한개의 요소만 빠져있다.
빠져있는 아이템 1개가 무엇인지 찾아야한다.

N은 10만 이하
"""

def solution(A):
    N = len(A)
    original_array_size = N + 1
    
    items = set(range(1, original_array_size+1))
    for i in A:
        if i in items:
            items.remove(i)
    
    return items.pop() # 남은 항목은 1개이므로 pop(랜덤 뽑기 함수)도 가능
```
