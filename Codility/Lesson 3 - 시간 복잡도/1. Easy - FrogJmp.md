"""

개구리가 길 건너로 이동하고자 한다.

현재 X에 위치해있고, Y 이상의 지점으로 가야한다.

개구리는 항상 고정된 D 만큼 점프할 수 있다.

목적지에 도달하기 위해 필요한 최소 점프 횟수를 구해야한다.

X, Y, D는 1부터 10억 이하
"""

def solution(X, Y, D):
    need_jump = Y - X

    i, j = divmod(need_jump, D)
    if j > 0:
        i += 1
    return i
