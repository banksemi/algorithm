"""
개구리가 강 너머로 가려고하는데 처음에는 
position0 에 있고 X+1로 가고자함.

A는 X까지의 숫자들이 있고, 1부터 X까지 가득 채워졌을 때 X+1로 이동이 가능함.

N, X = 1~10만 
A 의 원소 -> 1부터 X 까지만 있음
"""

def solution(X, A):
    items = set()

    for i, x in enumerate(A):
        items.add(x)
        if len(items) == X:
            return i

    return -1
