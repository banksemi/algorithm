```
"""
N 기반동안의 주식 가격이 A 배열에 나타나있다.

P에 주식을 사서 Q에 팔면 이익은 A[Q] - A[P]
- 0 ≤ P ≤ Q < N

한번의 거래로 얻을 수 있는 최대 수익 반환

단순히 min-max로 접근하면 -> P<=Q 조건을 만족하지 못할 수 있다.

expect_max_prices[i] = i, i+1.. 날짜 등을 모두 고려했을 때 가장 비싸게 팔 수 있는 금액
for i in range(0,len(A)-1): # 마지막날엔 팔 수 없다
    i+1날부터 팔 수 있다.
    
주의:
- 이익을 얻을 수 없으면 0 리턴
"""

def solution(A):
    N = len(A)
    
    # A가 2개 미만이면 애초에 수익을 낼 수 없다.
    if N < 2:
        return 0

    expect_max_prices = [A[-1]] * N
    for i in range(N-2, -1, -1):
        expect_max_prices[i] = max(A[i], expect_max_prices[i+1])

    answer = 0
    for i in range(0, N-1): # 한칸 전까지만 살 수 있다.
        expect_profit = expect_max_prices[i+1] - A[i]
        answer = max(expect_profit, answer)
    
    return answer
```
