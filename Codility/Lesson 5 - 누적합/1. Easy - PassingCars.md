- 1_000_000_000 초과 조건

```
"""
non-empty array A 

A는 도로 위에 차량을 의미한다.
- 0이면 오른쪽, 1이면 왼쪽으로 이동중

0<=P<Q<N, P가 오른쪽, Q가 왼쪽으로 이동하는 것이 = 차량이 지나가는 것을 의미.
N = 10만

"""

def solution(A):
    # Implement your solution here
    result = 0
    count = 0
    for i in A:
        if i == 0:
            count += 1
        else:
            # 이전 차량과 조합을 만들 수 있음
            result += count
    
    return result if result <= 1_000_000_000 else -1
```
